📄 Integration Guide

Add VideoCap to any website in 3 steps. No npm install. No build step.

STEP 1 — Load the widget script

<!-- Add before </body> --> <script src="https://your-videocap-server.com/widget.js"></script>

STEP 2 — Add a container element

<div id="videocap"></div>

STEP 3 — Initialise with your API key

VideoCap.render({ container: '#videocap', apiKey: 'vc_YOUR_API_KEY', apiUrl: 'https://your-videocap-server.com', theme: 'light', // 'light' | 'dark' mode: 'options', // 'options' (buttons) | 'text' (free input) onSuccess: function(token) { // Enable your form submit button console.log('Verified ✓', token); }, onError: function(err) { console.error('Failed', err); }, });

SERVER-SIDE — Verify the token in your backend (recommended)

# Python (Flask / FastAPI / Django) import requests res = requests.post( 'https://your-videocap-server.com/api/captcha/validate', headers={'X-API-Key': 'vc_YOUR_API_KEY'}, json={'token': token_from_frontend, 'answer': answer_from_frontend} ) data = res.json() if not data['success']: abort(403, 'CAPTCHA failed')
